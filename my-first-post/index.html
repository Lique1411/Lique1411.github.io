introduction

this server is running on github’s codespace . I’ll try to solve this problem that can’t open the website by access 0Hzc.github.io

1.step1

get ready for the environment,to deploy this server need go , git ,and install hugo_extrended . Notice that hugo edition is hugo_extrended not the standard hugo : if use the standard hugo,you will can’t deploy other themes.

2. step2

according to the docs from hugo’s official websites,use the following command:

hugo server
